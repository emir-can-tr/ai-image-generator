# AI Image Generator

A modern web application that generates images using artificial intelligence. Supports both **OpenAI** and **Google Gemini** APIs.

## Available Versions

This project is available in two languages:

| Version | Folder | Requirements |
|---------|--------|--------------|
| **Python** | [/python](./python) | Python 3.8+, Flask |
| **PHP** | [/php](./php) | PHP 7.4+, cURL extension |

## Features

- Modern, responsive interface with Glassmorphism design
- **OpenAI** and **Google Gemini** image generation support
- Multiple image size options (1024x1024, 1280x720, 720x1280, 1216x896)
- One-click image download
- Real-time loading animations
- Mobile-friendly design

## Quick Start

### Python Version

```bash
cd python
pip install -r requirements.txt
# Configure API keys in app.py
python app.py
```

### PHP Version

```bash
cd php
# Configure API keys in api.php
php -S localhost:8000
```

## Getting API Keys

### Google Gemini
1. Go to [Google AI Studio](https://aistudio.google.com/apikey)
2. Create a new API key
3. Copy and paste into configuration

### OpenAI
1. Go to [OpenAI Platform](https://platform.openai.com/api-keys)
2. Create a new API key
3. Copy and paste into configuration
